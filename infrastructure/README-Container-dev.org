* Arbeiten mit Podman/Podman-Compose
** Stoppen
*** Anhalten: Stoppen 
#+begin_src bash

  cd infrastructure
  # podman-compose down
  podman-compose -f podman-compose.prod.yml -f podman-compose.dev.yml down

  # if production

  podman-compose -f podman-compose.prod.yml down
  
#+end_src

*** Anhalten: Stoppen und Volumes löschen
Falls alles frisch aufzusetzen ist:

#+begin_src bash

  podman-compose down -v

#+end_src

(-v entfernt auch die Volumes, also deine PostgreSQL-Daten!)
** Löschen: Altes Image

#+begin_src bash
  
  podman images
  podman image rm <image_name_or_id>
    
#+end_src
** Löschen: <none> - Images
#+begin_src bash
  podman image ls -a | grep '<none>' | awk '{print $3}' | xargs podman rmi -f
#+end_src
** Löschen: Tailwind-Image

#+begin_src bash

  podman volume ls                  # Liste aller Volumes
  podman volume rm tailwind_build

  # Danach Dev-Container neu starten:

  podman-compose -f podman-compose.prod.yml -f podman-compose.dev.yml up
  
#+end_src

** Neu bauen: Podman-Compose


Alles neu bauen ...

#+begin_src bash

  podman-compose build --no-cache
    
#+end_src

** Starten:
*** Podman-Compose -- development

#+begin_src bash
    # 1. Dependencies installieren (einmalig / CI)
    bun install

  # 2. Tailwind CSS im Watch-Mode starten
    bun run watch:css
  
    # 3. Container starten
    cd infrastructure
    podman-compose -f podman-compose.prod.yml -f podman-compose.dev.yml up

#+end_src

*** Podman-Compose -- production

#+begin_src bash

  # 1. Fertige CSS bauen
  bun run build:css

  # 2. Docker Image bauen
  podman build -t mantis_web ..

  # 3. Container starten
  podman-compose -f podman-compose.prod.yml up -d
  podman-compose -f podman-compose.prod.yml up -d --build

#+end_src
