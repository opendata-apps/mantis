Benutzeroberfläche
==================

Dieses Kapitel beschreibt die produktiven Benutzerwege der Weboberfläche.

Abgedeckte Rollen
-----------------

- Melder (öffentlich): Meldungen erfassen, Verlauf einsehen, Karte nutzen
- Reviewer (Rolle ``9``): Meldungen prüfen, korrigieren, exportieren

Terminologie
------------

- ``Sichtung``: Beobachtung in der Realität
- ``Meldung``: gespeicherter Datensatz zu einer Sichtung
- ``Melder``: Person, die eine Meldung einreicht
- ``Finder``: Person, die das Tier gefunden hat (optional abweichend)
